# Sales Performance Dashboard (Excel)

## Overview
This project is an interactive Sales Performance Dashboard created in Microsoft Excel. It provides insights into sales performance, customer trends, product performance, and regional sales using Pivot Tables, Pivot Charts, KPIs, and Slicers.

## Dashboard Features
- Total Sales
- Total Orders
- Average Order Value
- Average Quantity Ordered
- Sales by Product Line
- Sales by Country
- Yearly Sales Trend
- Monthly Sales Trend
- Sales by Deal Size
- Orders by Status
- Top 10 Customers by Sales
- Sales by Territory

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

- Data Cleaning

- Sales Performance Analysis
